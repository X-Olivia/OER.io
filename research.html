My Research

Overview, paper, code, and results.

Overview

This project develops a physiology-informed, data-driven framework for MC and ovulation inference from sparsely sampled physiological signals, and integrates it into an embodied menstrual service robot. The methodology treats the MC as a temporally structured latent process: day-level physiological features (e.g. from temperature and cardiac signals) are combined with ovulation anchoring from urinary LH. A sequential multi-task model supports both ovulation-centred cycle state inference and menstrual onset prediction. Current status: literature review, data preprocessing, ovulation annotation pipelines, and baseline reproduction (DT, RF, LSTM, hierarchical generative, Poisson) under subject-level evaluation; robotic platform (ROS, pose estimation, SLAM, navigation) implemented as a supporting demonstrator.

Paper

Interim Report: An Embodied Care Robot Supported by Physiological Menstrual Cycle Inference. Jing Xu; Supervisor: Professor Dave Towey. The report covers background and motivation, literature review (physiological constraints, computational approaches, performance limits), methodology (data processing, sequential multi-task latent cycle state model, evaluation design), implementation (preprocessing, ovulation annotation, baseline reproduction, robot system), and progress reflection.

Abstract: MC inference is challenging due to variability, sparse observations, and indirect proxies; in embodied healthcare it acts as an upstream component for proactive robotic decision-making. This project investigates a framework based on sparsely sampled physiological signals and its integration into an embodied menstrual service robot. The methodology uses day-level physiological features and LH-based ovulation anchoring; a sequential multi-task strategy supports ovulation-centred state inference and menstrual onset prediction. Completed: literature review, preprocessing, ovulation annotation, baseline reproduction, and robotic system architecture. Future work: end-to-end integration of wearable-derived features, evaluation under sparse/irregular regimes, and closed-loop robotic behaviours.

Code & reproducibility

Implementation includes: data preprocessing (hormonal alignment, self-reported cycle structure); ovulation annotation module (LH surge onset/peak/final peak + 1 day, peak detection, multi-peak handling); baseline models (DT, RF, LSTM on self-reported/sequence inputs; hierarchical generative and Poisson on cycle length). Robot: ROS-based modular service robot (pose estimation, SLAM, TEB navigation). Code repository and run instructions will be linked here.

Results & figures

Preliminary baseline performance (subject-level splitting; MAE in days): Self-reported multi-modal (DT ≈ 1.87, RF ≈ 1.90); self-reported sequence LSTM (W=3) ≈ 1.94; historical cycle length—hierarchical generative ≈ 3.46, Poisson ≈ 3.45. These verify implementation and baseline behaviour; final model evaluation and error–day-in-cycle analysis are part of ongoing work. Figures from the report (e.g. LH trajectories with ovulation anchors, model structure, timelines) can be added to this section or linked from the paper.
