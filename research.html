Project focus

This project studies menstrual event prediction in a daily setting using wearable physiological data, calendar history, and limited personalisation. The current emphasis is not only on whether wearables help on average, but on when they help, and for which forms of irregularity.

A central claim of the current work is that irregularity should not be treated as one coarse label. Longer cycles, high-variability cycles, and shifted-but-stable cycles may create meaningfully different prediction problems.

Why this matters

The field often asks whether wearables work. A more precise question is where they add value beyond personal history and whether that value survives realistic prediction constraints. The present project tries to answer that narrower and more defensible question.
