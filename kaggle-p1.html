Kaggle Dataset Analysis

A local analysis page for the Kaggle menstrual cycle dataset. It summarises the dataset, the cleaning and feature-engineering decisions, and the baseline experiments I ran on it.

Overview Files Pipeline Results Limitations

Overview

This local analysis focuses on preparing the Kaggle menstrual cycle dataset for modelling, especially for cycle-length prediction under grouped evaluation. The workflow cleans the raw table, engineers a medication-impact feature, removes or imputes high-missingness variables using domain logic, and then evaluates three baseline model families: Decision Tree, Random Forest, and LSTM.

What the analysis does

  1. Checks the raw Kaggle table and inspects missingness in cycle and fertility variables.
  2. Selects candidate features for modelling rather than using the full raw schema unchanged.
  3. Processes medication text into a structured Med_Class feature with domain-based classes.
  4. Uses participant-level mean imputation for selected cycle variables such as ovulation day and menses length.
  5. Builds a compact modelling table with participant ID, cycle number, core reproductive-history variables, ovulation estimate, and target cycle length.
  6. Evaluates Decision Tree, Random Forest, and LSTM baselines under grouped validation to reduce leakage.

This matters because menstrual datasets often contain many sparse demographic or clinical columns that look informative but are too incomplete to use naively. This analysis makes those feature decisions explicit.

In the local documentation, EstimatedDayofOvulation is reported as the dominant predictor, followed by LengthofMenses, CycleWithPeakorNot, and ReproductiveCategory.

Limitations and interpretation

This dataset is useful for baseline experimentation, but it should not be mistaken for a modern wearable physiology dataset. It is much stronger for cycle-structure analysis than for low-level physiological signal modelling.

The value of this page is to show what the dataset looks like after cleaning, which features were retained, and what performance simple baseline models achieved under grouped evaluation.
