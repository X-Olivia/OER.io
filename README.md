# OER — Menstrual Cycle Prediction Research

Academic, minimal static site. Black, white, gray, blue, metallic palette. English. Content placeholders: `(keep working)`.

## Run locally

Open `index.html` in a browser, or use a local server:

```bash
# Python 3
python3 -m http.server 8000

# Node (npx)
npx serve .
```

Then visit `http://localhost:8000`.

## Structure

- `index.html` — Overview
- `survey.html` — Field survey (background, taxonomies, methods, limits, hotspots)
- `resources.html` — Datasets & data-processing code
- `research.html` — My research (overview, paper, code, results)
- `about.html` — About this OER & author
- `license.html` — License & citation
- `css/style.css` — Global styles

Replace all `(keep working)` placeholders with your content.
