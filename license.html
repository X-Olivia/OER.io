License & Citation

How to reuse this site responsibly, and how to cite it.

License status

A final public license has not yet been declared in this repository. Until you choose one, treat the original site text and layout as reusable only with attribution and with the author's permission for substantial republication.

Third-party papers, datasets, and external figures linked from this OER keep their own licenses and terms. Reusing those materials requires checking the original source, not only this site.

If you want this website to function as a formal open educational resource in the public sense, a permissive content license such as CC BY 4.0 would usually be the cleanest next step. That decision, however, should be made explicitly rather than implied.

Suggested citation for this OER

Temporary suggested citation format until a permanent public URL is fixed:

Xu, Jing. Open Educational Resource for Menstrual Cycle Prediction Research. 2026.
Website. URL to be inserted on public release.

Suggested BibTeX entry:

@misc{xu2026oer,
  author       = {Xu, Jing},
  title        = {Open Educational Resource for Menstrual Cycle Prediction Research},
  year         = {2026},
  howpublished = {Website},
  note         = {Add public URL on release}

How to cite external sources used here

  • Cite the original paper when discussing a study result.
  • Cite the dataset landing page or dataset paper when reusing data.
  • Cite both the OER and the underlying source when this site helped you locate or interpret the material.
